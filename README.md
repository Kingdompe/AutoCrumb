# AutoCrumb — Auto Delete Cookies on Tab Close

A modern Chrome extension that automatically deletes cookies when you close tabs. Built from scratch for Manifest V3 as a replacement for Cookie AutoDelete.

## Features

- **Auto-delete cookies on tab close** with configurable delay
- **Unlimited whitelist** — protect sites you trust (no slot limits)
- **Greylist** — temporary protection for shopping/banking sessions
- **Wildcard rules** — `*.google.com` covers all Google services
- **Cookie count badge** with color-coded status
- **Right-click context menu** for quick whitelist/greylist
- **Import from Cookie AutoDelete** — migrate your rules in seconds
- **Cloud sync** — rules sync across all your Chrome devices
- **Keyboard shortcuts** — Alt+Shift+C (toggle), Alt+Shift+W (whitelist), Alt+Shift+X (clean)
- **Optional site data cleanup** — LocalStorage, IndexedDB, Cache, Service Workers

## Install

Install from the [Chrome Web Store](https://chrome.google.com/webstore) (link coming soon).

### Developer Install

1. Clone this repository
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder

## Privacy

AutoCrumb is 100% local. No external servers, no analytics, no data collection. Your browsing data never leaves your machine.
